# SacredClips – User Guide

This guide explains how to use **SacredClips** to generate short educational videos about religious or spiritual topics.

SacredClips automatically creates:

- a script
- scene images
- voice narration
- a final vertical video

The video can then be previewed and downloaded for posting on platforms like **TikTok, Instagram Reels, or YouTube Shorts**.

---

# 1. Starting the application

Before using SacredClips, make sure both the **backend** and **frontend** are running.

### Start the backend

From the project root:

```bash
source .venv/bin/activate
uvicorn app.main:app --reload
```

Backend runs at:

```
http://localhost:8000
```

---

### Start the frontend

Open another terminal:

```bash
cd frontend
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

Open this address in your browser.

---

# 2. Generating a video

When the frontend loads, you will see a form with several fields.

### Step 1 — Enter a topic

Enter a **religious or spiritual topic**.

Examples:

- What is Ramadan?
- What is the meaning of baptism?
- What is the Sabbath in Judaism?
- What is Diwali in Hinduism?

SacredClips is designed for **educational religious topics** only.

---

### Step 2 — Choose narration style

You can modify the tone of the video.

Examples:

- neutral explainer
- calm educational tone
- respectful documentary style

---

### Step 3 — Select the duration

Choose the desired video length using the duration slider.

Recommended range:

```
60–90 seconds
```

Shorter videos work best for social media platforms.

---

### Step 4 — Generate the video

Click:

```
Generate Video
```

The system will now create the video.

---

# 3. What happens during generation

SacredClips performs several steps automatically:

1. A **script** is generated for the topic.
2. The script is divided into **multiple scenes**.
3. AI generates **images for each scene**.
4. AI generates **voice narration**.
5. The system combines images and narration into a **vertical video**.

This process usually takes **10–40 seconds**, depending on system speed and API response time.

---

# 4. Viewing the results

After generation finishes, the interface will display:

### Script

The full narration script used for the video.

---

### Scene breakdown

Each scene includes:

- scene text
- keywords used for image generation
- scene duration

---

### Video preview

A built-in player allows you to watch the generated video directly in the browser.

---

# 5. Downloading the video

Below the video preview you will find a **Download MP4** button.

Click the button to download the final video file.

The video can then be uploaded to platforms such as:

- TikTok
- Instagram Reels
- YouTube Shorts

---

# 6. Editing the script

If you want to change the content of the video:

1. Edit the script text in the interface.
2. Remove or modify sections you want to change.
3. Click **Regenerate Video**.

The system will regenerate:

- narration
- scene images
- the final video

using your edited script.

This allows you to refine the message or tone of the video.

---

# 7. Demo mode vs AI mode

SacredClips can run in two modes.

---

## Demo mode

If no OpenAI API key is configured:

- placeholder images are used
- narration may be silent
- scripts are simplified

This mode is useful for development or testing.

---

## AI mode

If an **OpenAI API key** is configured:

- scripts are generated by AI
- images are AI-generated
- narration is generated with AI voices

This produces higher-quality videos.

---

# 8. Where generated videos are stored

All generated content is saved in the project folder:

```
outputs/
```

Inside this directory you will find:

```
outputs/
   topic_name/
      images/
      audio/
      videos/
```

The final video file is usually located at:

```
outputs/.../videos/final_video.mp4
```

# 9. Troubleshooting

### Video preview does not play

Make sure:

- the backend server is running
- the frontend is connected to `localhost:8000`

---

### Images are placeholders

This usually means:

- OpenAI image generation failed
- an API key is missing
- the API request returned an error

Check backend logs for details.

---

### Narration missing or silent

Possible causes:

- OpenAI TTS not configured
- invalid API key
- API usage limit reached

---

### Video generation fails

Check the backend console for errors related to:

- MoviePy
- FFmpeg
- OpenAI API

---

# 10. Responsible use

SacredClips generates **AI-created educational material** about religion.

Before publishing videos:

- review the content carefully
- verify factual accuracy
- ensure respectful representation of traditions

The system should be used as a **content creation assistant**, not as an authoritative source.
